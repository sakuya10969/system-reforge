# プロダクト概要

## 対象

COBOLなどのレガシーシステムのソースコード。

## 目的

レガシーコードを「読んで理解する」ツール。以下を実現する：

- ソースコードの構造解析
- 業務ロジックの抽出
- 要件定義書の生成支援
- モダナイゼーション計画の策定支援

## これは何でないか

- コード変換ツールではない（COBOL → Java のような変換はしない）
- リファクタリングツールではない
- コードを書き換えるツールではない

## 変換の方向性

```
コード → 構造/意味の中間データ → 要件定義
```

ソースコードをそのまま要件にするのではなく、構造化された中間データを経由して意味を抽出する。

## 想定機能

### ZIPアップロード
- レガシーコードをZIPファイルでアップロード
- アップロード後、S3に原本保存

### 解析ジョブ
- アップロード後に解析ジョブを作成
- ワーカーが非同期で解析を実行
- 解析結果は構造化データとしてDB保存

### 依存関係・フロー可視化
- プログラム間の呼び出し関係をグラフ表示
- 処理フローの可視化

### 業務ルール抽出
- 条件分岐・計算ロジックを業務ルールとして抽出
- LLMを使って自然言語の業務ルール記述を生成

### 要件レビュー
- 抽出した業務ルールを要件定義形式で表示
- レビュー・編集・エクスポート機能
